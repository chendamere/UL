Universal Language

Universal Language is a starting point for machine thinking. It is a language that describes the process of code executions and rule inferences in a multi-dimensional cyclic datastructure. Through Universal Language (UL), a program can define concepts, and explain the principles of the language itself and also mathematics.

The theory is developed by Weili Chen and the website is maintained by Hankai Chen. The first volume of UL is publish and viewers can access a free version of the original PDF below, or view through the online version (in progress).
