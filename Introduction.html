Data Structure

Universal language is designed to describe and manipulate data structure that qualifies as tree-like multidimensional structure with doubly linked list in each dimension.

Node

The basic element of data structure is node. A node consists of data value, unique node id, and links pointing to the next node, previous node, and child node.
